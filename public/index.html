Manual Synchronization Steps

Step 1: Check Current Status

Run git status to see uncommitted changes

Run git branch to confirm current branch

Step 2: Commit Local Changes

If you have changes: git add [files] then git commit -m "message"

Keep commits focused and descriptive

Step 3: Push to Remote

Push your branch: git push origin [branch-name]

This makes your changes available to other environments

Step 4: Pull on Other Environments

SSH to other machines and run: git pull origin [branch-name]

For DigitalOcean: Be careful with production changes
